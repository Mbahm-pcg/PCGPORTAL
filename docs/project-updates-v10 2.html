7-Phase Pipeline

Every project flows through a structured pipeline — each phase has its own checklist, notes, and file uploads.

  • Zoning — classification, variance, hearing date, approval
  • Land Development — surveys, stormwater, plan filings (skippable)
  • Permitting — building, electrical, plumbing, mechanical permits
  • Pre-Construction — site plans, drawings, PKL approval
  • Equipment Ordering — DCP, NCR/POS, DMB, headsets, cameras
  • Construction / Install — delivery dates, install schedules, contractors
  • Completion — final walkthrough, Dunkin' approval, grand opening

Quick-Nav Tiles

Project detail pages can get long. Now there's a row of phase tiles at the top — click any one to jump straight to that section.

  • Shows completion percentage for each phase at a glance
  • Skipped phases appear dimmed (e.g., Land Development)
  • Inspections tile shows total count with its own red accent
  • Smooth scroll animation takes you right to the section
  • Always visible at the top of the Checklist tab

Inspections Tracker

Inspections now have their own dedicated section — no longer buried inside the Completion phase. Track every inspection with full context.

  • 11 inspection types: Building, Fire, Health, Electrical, Plumbing, Mechanical, Final, C of O, Rough-In, Framing, ADA
  • Results: Passed, Failed, Conditional, Scheduled, Pending
  • Notes field per inspection — record findings, corrections needed
  • File attachment per inspection — upload photos, PDFs, reports
  • View attachments inline with the full document viewer
  • Color-coded borders: green = passed, red = failed, yellow = pending
  • Header badge shows pass rate (e.g., "2/4 Passed")
📁

Large File Support

Upload architectural drawings, permit sets, and construction documents of any size — the system automatically chunks files to work within serverless limits.

  • Supports files up to 48 MB (previously capped at ~4.5 MB)
  • Automatic chunking: splits base64 into 4 MB segments
  • Seamless reassembly on download — you never see the chunks
  • Backward compatible with all previously uploaded files
  • Works across Plans & Drawings, Permits, and Inspections
  • Upload progress feedback with file size confirmation

Inline Document Viewer

View uploaded documents right inside the portal — PDFs render in a full-screen overlay, images display at full resolution. No need to download first.

  • Full-screen modal overlay with dark backdrop
  • PDF rendering via blob URLs — handles files of any size
  • One-click download button for saving to your device
  • Shows filename, file size, uploader name, and upload date
  • Images display with contain-fit scaling
  • Click outside or press ✕ to close

Contractor Roster

Save your regular contractors once, reuse them on every project. Manage the roster in Settings, auto-fill on project pages via dropdown.

  • Four contractor types: GC, Plumbing, Electrical, Mechanical
  • Save from project: "Save to Roster" button on any contractor card
  • Auto-fill: select a saved contractor from dropdown on any project
  • Manage in Settings → Vendors → Contractors tab
  • Stores company name, contact person, phone, and email
  • Cloud-synced — available across all sessions and users

Philadelphia City Data

Pull live data from 6 city APIs for any Philadelphia address. Zoning, property records, permits, violations, 311 complaints, and crime data — all in one click.

  • AIS address resolution — normalizes addresses to city format
  • Property data: market value, sale history, year built, zoning
  • Business licenses: active, expired, type, issue dates
  • L&I violations: code violations, status, resolution dates
  • 311 service requests: complaints within 0.25 miles, last 6 months
  • Crime incidents: Part 1 & 2 crimes within 0.25 miles, last 90 days
  • L&I appeals: hearing dates, decisions, project descriptions
  • Auto-populates zoning info on project detail pages
